Responsive Redesign

for Yan's Cuisine Home Page

Project Background

The purpose of this project is to examine an existing webpage, identify problems associated with usability and accessibility, and then develop a responsive, redesigned version.
I chose Yan's Cuisine's home page as the subject for redesign, and you can find its photos in the section below. The project will walk you through my design process, and you will find a redesigned webspage at the end.

Design Choices

Layout for Larger Containers

The body is divided by grid, because the design is more static and follows a simple verticle flow regardless of screen size.
Each grid has the flex layout, because it is easier to arrange smaller elements and allow for responsive adjustments.

In-between Sizes

Most elements have units of "vw" or in percentage, which allows for automatic adjustment to change in screen size.
The breakpoint in screen size is chosen such that when elements do change size (Ex. h1 changes from 8 to 9 to 11vw for larger, medium, and small screen), the size increase wouldn't be too drastic.

Interactive components

The links in header, footer, and the buttons are interactive. The color changes on hover and click.
